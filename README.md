# MMZ-Proof-Creator
A (mostly vibe-coded) locally-hosted website for creating Metamath Zero proofs.

## How to Run
Requirements
  - python3
  - Fira Code font
  - A browser
Run `python3 server.py`

## How to Use
Should be simple enough. The "Proof" section in "Add Proof" allows for the hypotheses to be referenced by number, 1-indexed (e.g. first hypothesis is `$1`)
